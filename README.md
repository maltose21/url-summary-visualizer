# URL Summary Visualizer

AI Agent Team 驱动的 URL 内容可视化工具。将任意链接转换为精美的单屏 HTML 总结页面。

## 功能特性

- **4 角色 Agent Team 流水线**：主编 · 内容分析师 · 图表设计师 · 布局设计师
- **全程可见可干预**：用户可查看每步产出、审核结论、驳回报告
- **自进化记忆系统**：三角色各自积累经验，越做越专业
- **纯 SVG 图表**：不依赖任何外部 CDN，任意环境均可渲染
- **Anthropic + Apple 视觉风格**：橙色强调 + 圆角卡片 + 柔和阴影

## 流水线

```
Step 1: 内容分析（内容分析师）
Step 2: 图表设计（图表设计师）
Step 3: 布局决策（主编）
Step 4: HTML 生成（布局设计师）
Step 5: 主编审核（可驳回修正）
Step 6: 交付
```

## 自进化记忆库

```
~/.url-summary-visualizer/
├── memory/
│   ├── analyst/        # 内容分析师记忆
│   ├── designer/       # 设计师记忆
│   ├── editor/         # 主编记忆
│   └── shared/         # 跨角色共享
└── workspace/          # 任务工作区
```

## 使用方式

在 Claude Code 中触发：

```
对这个链接生成一屏总结：https://example.com
```

或提及以下关键词：
> "总结链接"、"一屏总结"、"页面可视化"、"生成可视化"

## 适用场景

- 文档/文章快速总结
- 技术文档可视化
- 新闻/报告一页概览
- 工具/产品介绍页提炼
